## Arch Linux Docker 
### Dockerfile builds a archlinux:latest image with gcc, clang, rust, and a lightweight linux env.

## Start/Build Container:
- `$ docker compose build && docker compose up`
## Attach to Containers Shell:
- `$ docker attach $ARCH-DEV-N`
